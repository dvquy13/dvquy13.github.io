# DvQ's Personal Blog Site

This repo hosts the source code for DvQ's personal blog site, built using Quarto and deployed via GitHub Pages.

## Instructions

To render preview at local:
- Install QuartoCLI
- Run `quarto preview --port 8183`

To deploy changes:
- Push changes to `main` branch — GitHub Actions auto-renders and deploys to GitHub Pages.
- Do **not** commit `docs/` — it is gitignored and regenerated by CI on every push.
- GitHub Pages settings (Repo Settings → Pages): source branch `gh-pages`, folder `/`.
