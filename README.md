# DvQ's Personal Blog Site

This repo hosts the source code for DvQ's personal blog site, built using Quarto and deployed via GitHub Pages.

## Instructions

To render preview at local:
- Install QuartoCLI
- Run `quarto preview`

To deploy changes:
- Run `quarto render` to prepare `/docs` files
- Push the changes to `main` branch. The deployment is configured at Repo Settings > Pages.
