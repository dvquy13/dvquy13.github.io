# DvQ's Personal Blog Site

This repo hosts the source code for DvQ's personal blog site, built using Quarto and deployed via GitHub Pages.

## Instructions

To preview locally:
- Install QuartoCLI (version must match CI — see `.github/workflows/publish.yml`)
- Run `make run` (starts preview server at port 8183)

To build and validate:
- `make build` — version check → render → validate RSS feed
- `make validate` — validate RSS feed only (against local `docs/index.xml`)
- `bash scripts/validate-rss.sh --live` — validate the deployed feed at dvquys.com

To deploy changes:
- Push changes to `main` branch — GitHub Actions auto-renders, validates, and deploys to GitHub Pages.
- Do **not** commit `docs/` — it is gitignored and regenerated by CI on every push.
- GitHub Pages settings (Repo Settings → Pages): source branch `gh-pages`, folder `/`.
